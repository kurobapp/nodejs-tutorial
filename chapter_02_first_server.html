Chapter 2: Webサイトのフロントエンドを作成しよう

このチャプターでは、Node.jsサーバーが提供するWebサイトの見た目と動作を決めるHTMLファイルを作成します。

1. プロジェクトファイルを作成

Node.jsプロジェクトのファイルを制作しましょう。任意の場所（Cドライブなどのローカルのドライブを推奨）にNodeプロジェクトのファイルを制作してください。（このサイトでのファイル名はnodejs-tutorialにします。）

2. `index.html`ファイルを作成

さきほど作成した、プロジェクトファイルの中に、`index.html`という名前のファイルを作成し、以下のコードをコピー＆ペーストしてください。

このHTMLファイルは、メッセージ送信フォームを含むWebページの骨組みとなります。

`index.html` の内容:


<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Node.jsテストサーバー</title>
    </head>
<body>
    <main>
        <section>
            <h2>メッセージ送信フォーム</h2>
            <p>このフォームからサーバーへテキストメッセージを送信し、サーバーのコンソールログで確認します。</p>

            <h3>メッセージ送信</h3>
            <form id="messageForm" action="/submit-message" method="post">
                <label for="message">メッセージを入力してください:</label><br>
                <input type="text" id="message" name="message" value="Hello Node.js" size="40"><br><br>
                <input type="submit" value="サーバーに送信">
            </form>

            <p style="font-size: 0.9em; color: #666; font-style: italic; margin-top: 15px;">
                送信されたメッセージは、このNode.jsサーバーを起動しているターミナルに表示されます。
            </p>
        </section>
    </main>

    </body>
</html>
